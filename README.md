# Attune Phase 1

A calm, single-user Expo React Native + TypeScript starter for Attune.

## What is included
- very short Attune intro
- pre-session state check-in
- three session modes: Steady, Soften, Clear
- 3-minute breath-led solo session
- post-session reflection
- local session history only

## Run locally
1. Install Node.js 20+.
2. In this folder, run:
   - `npm install`
   - `npx expo start`
3. Open the project in Expo Go on your iPhone.

## Notes
- This build is deliberately Phase 1 only.
- No accounts, sync, AI, pair mode, or analytics are included.
- Session history is stored locally with AsyncStorage.

## Suggested next commands
- `npx expo install --fix`
- `npx expo start --clear`
